# CLAUDE.md

I am building a rapid prototyping assignment for a design class.

Project goal:
Create 25 landing page iterations for the same idea, plus 1 gallery page that shows the design process.

My idea:
A personal portfolio landing page for Yijin Wang.

Requirements:
- Use only pure HTML and CSS
- No frameworks
- No React
- No Tailwind
- No external APIs
- No backend
- No data persistence
- Keep everything simple and easy to deploy on Vercel

Structure:
- index.html = gallery page
- pages/v01.html to pages/v25.html = landing page iterations
- styles/gallery.css = gallery page styles
- styles/v01.css to styles/v25.css = styles for each version

Design process:
- Early versions should be radically different
- Middle versions should start combining stronger ideas
- Versions 20 to 25 should clearly converge toward a final direction

Style preferences:
- Clean, visually distinct iterations
- Strong typography
- Clear layouts
- Fast prototyping over perfection
- Keep the code simple and class-friendly
